# 刷机
进入不死uboot，使用固件名：```openwrt-mediatek-filogic-glinet_gl-mt6000-squashfs-sysupgrade.bin```丢进去刷入即可。
full = 全插件
docker = 带docker
slim = 精简版
beggar = 乞丐版
```
账号：root
密码：password
地址：192.168.11.1
```
- WiFi：WiFi配置已设置好，首次刷入开机等待几秒出来信号。
- WiFi信号：OpenWrt_2.4G/OpenWrt_5G 密码：无 可自己去设置
- 支持USB3.0 5G上网模块。
